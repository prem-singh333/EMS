# Event Managment System

## Project Overview

- This is a web-based Event Management System that allows user to explore, join events.
- Organizers can create and manage events while users can participate in them.


## Features

### User Features

- User Sign-up & Login
- View all available events
- Book events

### Organizer Features

- Create new events
- Update & delete events
- View all created events

### Tech Stack

- Frontend: React
- Backend: Laravel (API)
- Database: MySQL

### Authentication

- Token-based authentication is used
- Only logged-in user can book events
